# 🌍 WebQuest - Turismo Digital

## 📖 Descripción

**WebQuest - Turismo Digital** es una aplicación web educativa desarrollada con **HTML, CSS y JavaScript**, diseñada para apoyar el aprendizaje sobre el uso de las tecnologías digitales en el sector turístico.

La aplicación permite a los estudiantes explorar contenidos relacionados con páginas web, redes sociales, blogs y Tecnologías 4.0 aplicadas al turismo mediante una interfaz moderna e interactiva.

---

## 🎯 Objetivo

Desarrollar en los estudiantes competencias para analizar y utilizar herramientas digitales que contribuyan a la promoción de destinos turísticos, fomentando el trabajo colaborativo y el uso responsable de las Tecnologías de la Información.

---

## ✨ Características

- Diseño responsivo.
- Navegación mediante pestañas.
- Interfaz moderna con efectos CSS.
- Secciones organizadas por temas.
- Recursos digitales de consulta.
- Rúbrica de evaluación.
- Quiz interactivo.
- Preguntas de opción múltiple.
- Preguntas de verdadero y falso.
- Actividad de relación de columnas.
- Actividad de arrastrar y soltar (Drag & Drop).

## 🧠 Funcionalidades implementadas

### Navegación

Permite cambiar entre las diferentes secciones utilizando pestañas dinámicas.

### Evaluación

El sistema incluye:

- Calificación automática de preguntas de opción múltiple.
- Evaluación de preguntas Verdadero/Falso.
- Relación de columnas.
- Actividad Drag & Drop.

### Diseño

- Fondo con degradados.
- Tarjetas con sombras.
- Encabezado con imagen de portada.
- Barra de navegación fija.
- Animaciones suaves.
